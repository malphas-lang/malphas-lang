# Malphas Documentation

This directory contains documentation for the Malphas programming language project.

## Structure

- **Status & Planning**: Current status and future plans
  - `LLVM_STATUS.md` - Current status of LLVM backend
  - `LLVM_NEXT_STEPS.md` - Next steps for LLVM backend
  - `WORK_REMAINING.md` - Overall work remaining
  - `LLVM_BACKEND_PLAN.md` - Detailed LLVM backend plan

- **Language Design**: Language design documents
  - `VISION.md` - Project vision and goals
  - `malphas_pointer_model.md` - Pointer and reference model
  - `malphas_generics.md` - Generics design
  - `malphas_in_scope_safety_features.md` - Safety features

- **Implementation**: Implementation details
  - `RUNTIME_MIGRATION.md` - Runtime migration plan
  - `MODULE_SYSTEM_COMPLETION.md` - Module system documentation
  - `MODULE_SYSTEM_HANDOVER.md` - Module system handover
  - `PARSER_IMPROVEMENTS.md` - Parser improvements
  - `TYPE_SYSTEM_IMPROVEMENTS.md` - Type system improvements

- **Setup & Installation**: Installation and setup guides
  - `INSTALL.md` - Installation instructions
  - `INSTALL_EXTENSION.md` - VS Code extension installation
  - `INSTALL_PLUGIN.md` - Plugin installation
  - `LSP.md` - Language Server Protocol documentation
  - `LSP_SETUP.md` - LSP setup guide
  - `LSP_QUICKSTART.md` - LSP quickstart
  - `LSP_BACKGROUND.md` - LSP background information

- **Other**: Miscellaneous documentation
  - `EXECUTION_TARGET.md` - Execution target options
  - `FIX_SYNTAX_HIGHLIGHTING.md` - Syntax highlighting fixes
  - `handover.md` - Project handover notes
  - `implementation_plan.md` - Implementation plan
  - `plan.md` - General plan
  - `task.md` - Task list
  - `runtime_example.md` - Runtime examples
